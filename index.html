Temas de CSS

1. Definición de CSS:
CSS (Cascading Style Sheets) es un lenguaje de diseño gráfico utilizado para describir la presentación de un documento escrito en HTML o XML.
2. Funcionamiento Básico de CSS:
CSS funciona aplicando estilos a los elementos HTML para controlar su apariencia, como el color, la fuente, el espaciado, y la disposición en la página.
3. Incluir CSS en un archivo HTML:
CSS se puede incluir en un archivo HTML usando una etiqueta `style` dentro del `head`, un archivo externo enlazado con ``, o estilos en línea con el atributo `style`.
4. Definición de una regla CSS:
Una regla CSS define el estilo que se aplicará a los elementos HTML. Se compone de un selector y un bloque de declaración.
5. Elemento de una regla CSS:
  • Regla: Conjunto completo que define cómo se aplica el estilo.
  • Selector: El elemento HTML al que se aplica el estilo.
  • Bloque de declaración: Contiene una o más declaraciones CSS.
  • Propiedad: El aspecto del elemento que se va a cambiar.
  • Valor: La especificación de la propiedad.
6. Medios CSS:
Los medios CSS se usan para aplicar estilos específicos según el dispositivo o características del entorno, como `@media screen` para pantallas y `@media print` para impresoras.
7. Comentarios CSS:
Los comentarios en CSS se escriben entre `/*` y `*/`. Son útiles para dejar notas sobre el código.
8. Sintaxis de la definición de las propiedades CSS:
La sintaxis es `propiedad: valor;`, donde `propiedad` es el aspecto a modificar y `valor` es la configuración deseada, como `color: blue;`.
